## Known Results

- Verified for all n ≤ 10^17 (Salez 2014).
- Only primes p ≡ 1 (mod 24) need careful treatment.
- Mordell's polynomial identities cover n ≡ 2 mod 3, 3 mod 4, 2/3 mod 5, 3/5/6 mod 7, 5 mod 8.
- Combined: all n except n ≡ {1,121,169,289,361,529} mod 840 are covered.
- Natural density of exceptions → 0.

## Best Current Bounds

- Computational frontier: 10^17.
- Covering: 834/840 = 99.3% of residues mod 840 covered.
- Average solutions per prime: O((log n)^2).

## Key Mathematical Tools

- Covering systems of congruences
- Quadratic residue theory (Mordell's barrier)
- Hasse–Minkowski theorem (for each prime p, exists q with p non-QR mod q)
- Bombieri–Vinogradov theorem (Elsholtz–Tao)
- Egyptian fraction algorithms (greedy, Fibonacci)
