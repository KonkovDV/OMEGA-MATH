# Code of Conduct

## Our Commitment

We are committed to a respectful, harassment-free environment for everyone.

## Expected Behavior

- Be respectful and constructive.
- Focus on technical arguments, not personal attacks.
- Accept feedback gracefully.
- Help keep discussions inclusive and professional.

## Unacceptable Behavior

- Harassment, insults, or discrimination.
- Threats, intimidation, or doxxing.
- Deliberate disruption of discussions or review processes.

## Enforcement

Project maintainers are responsible for clarifying and enforcing this code of conduct.

Reported incidents may result in warnings, temporary restrictions, or removal from project spaces, depending on severity.

## Reporting

Report conduct issues privately to maintainers via GitHub.

For security issues, use the process described in SECURITY.md.
